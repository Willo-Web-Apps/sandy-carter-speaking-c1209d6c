# Design DNA — Carter Keynotes

## Visual Personality: dark-modern
## Layout: storytelling
## Typography: bold-display

### Color Palette
- Primary (gold accent): #D4A847 → oklch(0.73 0.14 80)
- Background (deep charcoal): #0F1117 → oklch(0.10 0.01 265)
- Text (warm white): #F5F0EB → oklch(0.96 0.01 80)
- Surface mid: #1A1D27 → oklch(0.14 0.015 265)
- Surface card: #20242F → oklch(0.17 0.015 265)
- Gold muted: #A07C2A → oklch(0.56 0.11 78)

### Font Pairing
- Display: Oswald — weight range 400–700 (condensed, commanding, stage presence)
- Body: Barlow — weight range 300–500 (modern, readable, corporate-clean)

### Unique Design Choices
1. **Cinematic hero with asymmetric spotlight** — Diagonal conic + radial gradients blend electric gold from the top-right corner, like a stage spotlight. Noise texture overlay gives it a film-poster quality, unlike any speaker bureau site.
2. **"SANDY CARTER" as monumental typographic anchor** — Name rendered in massive Oswald condensed caps; "CARTER" shifts to gold. Functions as both headline and background visual element. Instantly memorable.
3. **Credential marquee strip** — Immediately below hero, an animated horizontal strip of credibility markers (Davos, SXSW, Fortune 25, AWS, IBM, UN) in gold caps on charcoal destroys credibility doubt before it forms.
4. **Talks as premium dark cards with gold top-border** — Each keynote topic has a 2px gold top border, lifts on hover with a subtle gold glow — communicates expertise, not a generic bureau list.
5. **Testimonial isolation section** — A single powerful quote isolated in its own section with an oversized typographic gold quotation mark — forces the visitor to read it. No noise, no distraction.
